给自己留一个以后使用的备注：
1项目主页，下载zip包到桌面，解压；
2打开myeclipse，import引入-maven-已有的maven项目，选择解压后的文件夹；
3项目引入后，还需要进入WEB-INF/lib，把里面的jar包全选，右键-buildpath-add to build path；
  
  还有注意：已知jdk8是不行的，jdk7没问题，jdk版本不能选jdk8及以上；tomcat6是不行的，tomcat7可以，tomcat不能选择6及以下。
